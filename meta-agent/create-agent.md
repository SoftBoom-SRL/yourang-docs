# Create the Meta agent

Create the agent that will reply to messages and comments, choosing the right type and initial settings.

_With your accounts connected, you create the actual agent. The flow is guided and takes just a few minutes: choose the «Meta» type, give it a name, and decide how to handle comments._

## Open the agent creation flow <a id="apri-agenti"></a>

1. **Go to «AI Agents»** — In the Yourang side menu click the AI Agents section.
2. **Press «Create agent»** — You'll find the button at the top right of the agent list. A guided multi-step window opens.

## The creation steps <a id="passaggi"></a>

1. **Choose the «Meta» type** — On the first screen («Choose agent type») you will see several cards: Inbound, Outbound, and Meta. Click the «Meta» card — the one described as «Replies to Instagram and Facebook DMs and moderates comments». The selected card is highlighted.
2. **Press «Continue»** — You move to the details screen.
3. **Give the agent a name** — In the «Agent name» field type a recognizable name, for example «Reception» or «Social assistant». This is only for your own identification in the list. It must be at least 2 characters long.
4. **Decide whether to hide abusive comments** — You will find the toggle «Automatically hide abusive comments» (see below). It is on by default: leave it that way if you want the agent to hide spam and insults.
5. **Press «Create agent»** — The agent is created and its configuration page opens, where you will complete the settings (see the «Configure the agent» page).

## The «Hide abusive comments» option <a id="nascondi-abusivi"></a>

When this is enabled, the agent automatically hides comments that are clearly offensive or spam instead of replying. Important: genuine questions and legitimate criticism are NEVER hidden — they always receive a reply. The goal is to protect your profile's image, not to silence people who have something to say. You can change this setting later from the agent's settings.

## Right after creation <a id="dopo-la-creazione"></a>

> [!WARNING]
> **The agent is not yet active**
> As soon as it is created, the agent exists but does not reply yet: you must first complete the configuration (instructions, reply mode) and then activate it. You will find everything on the «Configure the agent» page.

> [!NOTE]
> **The type cannot be changed afterwards**
> A Meta agent remains a Meta agent: the type is chosen at creation and cannot be converted into a voice agent. If you need a different type, create a new agent.

# Connect Apple Calendar

Step-by-step guide to connect Apple Calendar (iCloud) to Yourang via CalDAV

_Connect your iCloud calendar to Yourang via CalDAV. You need your Apple ID and a dedicated app-specific password._

## Prerequisites <a id="prerequisiti"></a>

- An Apple ID with two-factor authentication enabled
- An app-specific password generated from your Apple ID

## How to connect Apple Calendar <a id="passaggi"></a>

1. **Open Apple ID settings** — Go to account.apple.com and sign in with your Apple ID.
2. **Generate an app-specific password** — Under Sign-In and Security → App-Specific Passwords, create a new password for Yourang.
3. **Enter the credentials in Yourang** — In Yourang → Integrations → Apple Calendar, enter your Apple ID and the app-specific password you just generated.
4. **Confirm and connect** — If needed, open advanced settings to change the CalDAV server or calendar name, then press Connect.

![Apple ID screen generating an app-specific password](/images/docs/integrations/apple-calendar/app-password.png)

_Where to generate the app-specific password in Apple ID settings_

> [!TIP]
> The app-specific password is different from your Apple ID password: it is a dedicated password you can revoke at any time.

> [!WARNING]
> Do not use your main Apple ID password: always use a dedicated app-specific password.

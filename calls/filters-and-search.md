# Filters and search

Quickly find the call you're looking for using filters by date, outcome, number, and keyword.

_When the call history grows long, finding a specific call can take time. Yourang provides a set of filters and a search bar to narrow down the list to the calls that matter to you._

## Available filters <a id="filtri-disponibili"></a>

Filters live at the top of the history. You can combine them freely to narrow results:

- **Date range** — Select a specific period (today, yesterday, last week, last month) or manually set a start and end date.
- **Call outcome** — Filter by outcome type: answered, abandoned, transferred, voicemail, unanswered. You can select one or more outcomes at the same time.
- **Phone number** — Enter a number or part of one to find all calls coming from that contact.
- **Voice agent** — If you use multiple Yourang assistants for different properties, you can filter calls by individual agent.
- **Keyword** — Search for a specific term within the summaries and transcripts of all calls in the selected period.

## Keyword search <a id="come-usare-la-ricerca-per-parola-chiave"></a>

Keyword search analyses the text content of summaries and transcripts. It's useful, for example, to find all the calls in which a guest's name, a room type, or a particular request such as "cot" or "parking" was mentioned.

> [!TIP]
> **Recommended workflow for daily review**
> Each morning, filter calls by "yesterday" and review the summaries of calls with the outcome "Transferred" or "Unanswered". These are the ones that might need a follow-up from your staff. In a few minutes you'll have a complete view of what needed attention while you were away.

## Resetting filters <a id="reimpostare-i-filtri"></a>

To return to the full view of the history, use the "Reset filters" button in the filter bar. All selected criteria will be cleared and the list will go back to showing all available calls.
